== Description ==

Simple plugin for changing WooCommerce's pages layout columns number.
Will also keep-make your WooCommerce pages responsive.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `woo-columns.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Will this make my shop pages responsive? =

Yes it will.

= Can i switch from 4 columns to 2,3 or even 1? =

Yes you can.
